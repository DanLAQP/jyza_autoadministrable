VIII. FLUJOS FUNCIONALES IMPLEMENTADOS ✅
================================================================================

1. GENERAR CERTIFICADO SIN USUARIO
   -------------------------------------------------
   Paso 1: Admin accede a /certificados/generar
   Paso 2: Ingresa DNI del titular
   Paso 3: Sistema busca titular (AJAX)
   Paso 4: Si existe → autocompleta nombres
   Paso 5: Si no existe → admin ingresa nombres manualmente
   Paso 6: Admin completa datos del curso
   Paso 7: Sistema crea/busca titular
   Paso 8: Guarda certificado con titular_id
   Paso 9: Certificado queda vinculado a DNI (sin usuario)
   
   ✅ RESULTADO: Certificado emitido sin crear usuario del sistema

2. CREAR USUARIO Y HEREDAR CERTIFICADOS
   -------------------------------------------------
   Paso 1: Admin accede a /users/add
   Paso 2: Selecciona rol "Estudiante"
   Paso 3: Ingresa DNI
   Paso 4: Sistema verifica titular (AJAX)
   Paso 5: Si titular existe → muestra certificados a heredar
   Paso 6: Admin completa datos de usuario
   Paso 7: Sistema vincula usuario con titular
   Paso 8: Usuario puede acceder a /certificados/misCertificados
   Paso 9: Ve certificados emitidos ANTES de crear cuenta
   
   ✅ RESULTADO: Usuario hereda certificados automáticamente

3. VERIFICACIÓN PÚBLICA
   -------------------------------------------------
   Paso 1: Empleador accede a /certificados/verificar
   Paso 2: Ingresa código CER-YYYY-XXXXXXXXXX
   Paso 3: Sistema busca certificado
   Paso 4: Carga datos del TITULAR (no usuario)
   Paso 5: Muestra datos inmutables (snapshot)
   Paso 6: Verificación válida sin importar cambios en usuario
   
   ✅ RESULTADO: Verificación robusta e independiente

4. ESTUDIANTE VE SUS CERTIFICADOS
   -------------------------------------------------
   Paso 1: Estudiante login
   Paso 2: Accede a /certificados/misCertificados
   Paso 3: Sistema obtiene users.titular_id
   Paso 4: Query: certificados WHERE titular_id = X
   Paso 5: Muestra TODOS los certificados del titular
   
   ✅ RESULTADO: Ve certificados previos + nuevos

5. ADMIN GESTIONA TITULARES
   -------------------------------------------------
   Paso 1: Admin accede a /titulares
   Paso 2: Lista todos los titulares con búsqueda
   Paso 3: Click en "Ver" → detalles + certificados
   Paso 4: Click en "Editar" → corregir nombres/DNI
   Paso 5: Eliminar solo si no tiene certificados
   
   ✅ RESULTADO: Gestión centralizada de identidades
